# Student-Performance-Analysis-System
This Python-based Student Performance Analysis System is a command-line application designed to manage student records, compute academic metrics, and generate visual performance insights. The project uses MySQL for secure data storage, Pandas for efficient data manipulation, and Matplotlib for creating interactive analytics charts.

├── Update existing student details
├── Delete student records
├── View all student records in a tabular format
├── Automatic Grade and GPA Calculation
